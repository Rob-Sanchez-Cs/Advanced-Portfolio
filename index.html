Here are some of my projects

  • Apple Website Clone

    Html, Tailwind CSS, JavaScript, React, GSAP, ThreeJS

    This project is meant to replicate the beautiful design strategy of Apple's flagship product page. Using GSAP, animations are replicated easily to look elegant yet simple and ThreeJS to create an amazing 3D model of Apple's flagship product, iPhone. Please enjoy this minimal yet stunning page in honor of Apple's amazing products.

  • Carepulse

    Tailwind CSS, TypeScript, NextJS, Appwrite, Shadcn

    A mock healthcare management system to simplify your healthcare plan. Using Appwrite for backend functionality, users can enter their healthcare information and book appointments with their physician of choice. Makes use of Shadcn's reusable components to simplify development and enhance user experience. Please enjoy this healthcare management clone.

  • Podcastr

    Tailwind CSS, TypeScript, NextJS, Convex, Clerk, OpenAI API

    Podcastr is an easy to use podcast platform. User authentication is managed with Clerk and backend functionality with Convex. With Podcastr, users can easily create and share podcasts. The real star of the show though is the utilization of OpenAI's API. Users can generate thumbnails using AI and even generate the audio using a text to speech model. Who doesn't love a good podcast?

  • OMDB

    Html, CSS, JavaScript, React

    A simple web application that allows users to search for movies, tv shows, and episodes. Utilizing the OMDB open API, media is displayed based on users search. Users can also filter based on release year and media type. Search for your favorite media!
